This summer I was given a very unique opportunity. One of my friends approached me and asked if I would like to work on an app for a special project that, for legal reasons, I might not be able to disclose. However I can say over the summer I was learning how to use Kotlin, a coding language used by Android to develop most of their apps. I haven't learned too much yet, only how to use functions and text commands, but there is so much more I can explore.

Datapack Creation

While learning Kotlin is an amazing experience, another project I am very passionate about is my map/datapack creation. I am in the works of creating an entire open-world RPG inside of minecraft, fully functional with custom villager trades, quests, items, mobs, bosses, etc. The progress so far has 2 bosses, 10 areas to explore, and a storyline to go with it. The build team and I are only about halfway done with the first part, and there is so much more content to add.
